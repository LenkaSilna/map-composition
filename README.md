# Map Composition

OpenLayers map application showing polygons with WMS background layer.

## Tech Stack

- React + TypeScript + Vite
- OpenLayers
- Proj4js

### Clone the repository

```bash
git clone [https://github.com/LenkaSilna/map-composition.git]
cd map-composition

```

### Install dependencies
```bash
npm install
```

### Development
```bash
npm run dev
```
